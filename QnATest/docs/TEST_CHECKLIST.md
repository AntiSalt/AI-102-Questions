## 🐛 常见问题排查

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 修改后点击多选题仍只能选一个 | 浏览器缓存 | Ctrl+Shift+R 强制刷新 |
| 答案选择后无法取消 | 脚本未加载 | 检查浏览器控制台（F12）是否有错误 |
| 提交后答案显示错误 | 脚本版本旧 | 清除LocalStorage：按F12→Application→清除Local Storage |
| 正确率计算不对 | 多选排序问题 | 刷新页面重新加载统计 |
