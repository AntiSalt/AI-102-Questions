# 🎓 AI-102 问答系统 - 快速参考

## 📌 系统信息

| 项目 | 说明 |
|------|------|
| **总题数** | 290题 |
| **题库来源** | Azure AI-102 考试题库 |
| **技术栈** | HTML5 + CSS3 + JavaScript (纯前端) |
| **数据存储** | LocalStorage (浏览器本地) |
| **响应式** | 完全支持移动端、平板、桌面 |

## 🚀 快速启动

### 步骤1：启动服务器

#### 方式A：使用 run.py 脚本（推荐）⭐
```bash
# 进入项目根目录
cd QnATest

# 运行启动脚本（会自动打开浏览器）
python run.py
```

#### 方式B：使用 Python 内置 HTTP 服务器
```bash
# 进入 src 目录
cd QnATest/src

# 启动 HTTP 服务器
python -m http.server 8000
```

#### 方式C：使用 Node.js (需要安装)
```bash
# 全局安装 http-server
npm install -g http-server

# 进入 src 目录
cd QnATest/src

# 启动服务
http-server -p 8000
```

### 步骤2：打开浏览器
- 如果使用 run.py，浏览器会自动打开
- 否则，手动访问：`http://localhost:8000`
- 如果改变了端口，使用对应的端口号

### 步骤3：开始答题
点击任何选项 → 点击"提交答案" → 查看结果 → 点击"下一题"

## 💡 核心功能一览

| 功能 | 位置 | 说明 |
|------|------|------|
| **侧边栏统计** | 左侧 | 实时显示进度、正确率等 |
| **题目显示** | 中央 | 背景、题目、4个选项 |
| **答案反馈** | 下方 | 正确/错误提示、解析 |
| **快速操作** | 侧边栏 | 查看统计、错题回顾、重新开始 |
| **题号跳转** | 侧边栏 | 输入题号跳转到任意题目 |
| **进度保存** | 浏览器存储 | 自动保存，刷新页面可恢复 |

## 🎯 答题流程

```
第1题显示
    ↓
点击选择答案（高亮）
    ↓
点击"提交答案"
    ↓
显示 → 正确/错误
     → 你的答案
     → 正确答案  
     → 详细解析
     → 争议说明（如有）
    ↓
点击"下一题"进入第2题
    ↓
重复...直到最后一题
    ↓
显示完成页面（正确率统计）
```

## 🔍 查看统计和错题

**方式1：点击侧边栏的"查看统计"按钮**
- 显示总题数、已答题、正确、错误、正确率
- 列出所有错题（题号、你的答案、正确答案）

**方式2：完成所有题目后**
- 自动显示完成页面
- 点击"查看错题"按钮

## 🔄 重新做题

**清除所有进度：**
1. 点击侧边栏"重新开始"按钮
2. 确认后所有答案将被清除
3. 从第1题重新开始

**做错题：**
1. 进入统计页面
2. 查看错题列表
3. 题号旁边暂时没有"重做"按钮，但可用题号跳转功能

## 💾 数据管理

### 查看当前进度
按 `F12` 打开浏览器控制台，输入：
```javascript
JSON.parse(localStorage.getItem('ai102_progress'))
```

### 导出进度为JSON
```javascript
const data = JSON.parse(localStorage.getItem('ai102_progress'));
console.log(JSON.stringify(data, null, 2));
// 右键选择"复制"
```

### 清除所有进度
```javascript
localStorage.removeItem('ai102_progress');
location.reload();  // 刷新页面
```

## ❓ 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| JSON无法加载 | 使用HTTP服务器方式打开（如上） |
| 进度丢失 | 检查浏览器是否禁用LocalStorage或处于隐私模式 |
| 看不到结果 | 滚动页面往下，或等待1秒让内容加载 |
| 想重新做某题 | 用侧边栏"题号跳转"功能 |
| 要导出进度 | 见上面"导出进度为JSON"部分 |
| 想改题库 | 编辑MD文件后运行`python parse_markdown.py` |
