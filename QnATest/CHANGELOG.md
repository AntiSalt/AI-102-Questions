## 版本说明

### 版本号格式
- **MAJOR** - 不兼容的API修改
- **MINOR** - 向后兼容的新功能
- **PATCH** - 向后兼容的bug修复

### 如何升级

1. **从旧版本升级到新版本：**
   ```bash
   git pull origin main
   # 或者重新克隆最新版本
   git clone https://github.com/用户名/QnATest.git
   ```

2. **清除旧的进度数据（如需要）：**
   ```javascript
   // 在浏览器控制台运行
   localStorage.removeItem('ai102_progress');
   location.reload();
   ```
